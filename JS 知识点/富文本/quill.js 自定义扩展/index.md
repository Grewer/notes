# quill.js 自定义扩展

## 前言

因为各类眼花缭乱的需求, quill.js 的编辑器也收到了各种挑战, 比如我们要添加 `table` 布局的样式来适配邮件的发送格式、
手动扩展 emoji 功能、图片的自由拖动等等, 本文就这些可定制化功能做一下讲解和实现

## 区分 format 和 module 

首先要明确自己需要的扩展是什么样子的?

比如我想要新增一个**自定义 emoji**, 那么想象一下步骤:

1. 点击工具栏
2. 弹出弹窗或者对应的 popover
3. 在 2 中选中 emoji

上述步骤就是一个常见的添加流程

首先我们需要确定的是, 添加自定义 emoji, 必然需要一个对应的格式
